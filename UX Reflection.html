UX Reflection

Working on this portfolio has taught me a lot about designing my own website. This is the first website project I've ever done so I am content with the amount of things I was able to experiment with. By using Bootstrap studio to create this project, I think I made it harder for me to include everything I wanted. However, I deliberately picked this method since I was interested in learning to use bootstrap studio since it seems that I will be using it next quarter for SE 2840 Web Dev so I wanted a headstart.

Design Rationale

For my rationale with designing this website, I wanted it to be clear for the user to find all the things that they want without too much confusion. This led me to try keeping minimal text on the navigations pages but when they go pages such as a specific project page, they can read more in depth the things I wrote.
